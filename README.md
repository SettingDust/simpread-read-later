## 🖥 功能

1️⃣ #keywords → 检索标签，包括：稍后读和标注的标签

2️⃣ keywords → 检索任意内容，包括：标题，描述，备注，标签，标注的备注，标签

3️⃣ 当打开这个稍后读后会优先打开它在本地的快照（如没有快照的话则直接打开原链接）

4️⃣ 搜索关键字以空格分割

## 📖 简悦

是一款集沉浸式阅读、标注、稍后读、导出等功能的新型一站式知识管理工具。

不保存用户数据，没有登录系统，极度注重用户隐私权。
